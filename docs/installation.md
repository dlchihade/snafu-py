# Installation Guide

## Prerequisites

- Python 3.5 or higher
- pip package manager

## Installation Methods

### From PyPI (Recommended)
```bash
pip install pysnafu
```

### From Source
```bash
git clone https://github.com/AusterweilLab/snafu-py.git
cd snafu-py
pip install -e .
```

## Verification

```python
import snafu
print(snafu.__version__)
```
